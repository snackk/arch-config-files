# tilda

### Description

My personal configs for the dropdown terminal tilda.

### Installation

To install ONLY this config file:
```sh
$ git https://github.com/snackk/config-files
$ cd tilda
$ ./install.sh && echo "Enjoy :)"
```
Any problem with install just run:
```sh
$ chmod +x install.sh
```
